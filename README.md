# Resolution-Game_Dev-Week-2

This game was made using Ren'Py for (Resolution Game Dev)[https://resolution.hackclub.com/app/pathway/game_dev], a [Hack Club](https://hackclub.com/) YSWS. It’s fully open source, and you’re free to use it however you like.

It’s a meta game about how I missed Week 2 by procrastinating too much and playing Clash Royale. It has four endings (bad, good, best, and secret), as well as three minigames (arrows, questions, and typing) and it also has CopyRight free music that i downloaded from the internet so credit to the maker that i don't remember his name.
